消息变动时的滚动处理
```
弄两个函数，分别是 获取收集数据 计算处理滚动
在会导致显示消息变化的地方，使用它这两个函数
变化前 调用 获取收集数据 将返回值接收为oldData
变化后 调用 计算处理滚动 传入刚刚的oldData

计算处理滚动
对比当前数据与旧数据，找出在顶部增加的，统计其高度，将当前滚动位置增加此高度
如果在顶部减少了，则当前滚动位置减少此高度
```
